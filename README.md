# Woofstagram 🐾
Este projeto foi desenvolvido utilizando Vite com React para a web, em vez de React Native, para facilitar a visualização imediata no navegador e agilizar o ciclo de desenvolvimento. Embora a plataforma seja diferente da sugerida no guia original, a lógica de navegação programática, o uso de hooks e a estrutura de rotas aninhadas seguem exatamente a mesma premissa exigida para o aplicativo móvel.

# Sobre o Projeto
O Woofstagram é um protótipo de rede social para pets. O objetivo principal foi consolidar conhecimentos sobre navegação e gerenciamento de formulários.

# O que foi implementado:
Navegação de Autenticação: Telas de Login e Cadastro utilizando rotas independentes.

- Navegação Principal (Main): Uma estrutura de "Tabs" (Abas) para alternar entre o Feed e o Perfil do usuário.

- Formulários: Cadastro de novos pets utilizando a biblioteca Formik.

- Componentização: Divisão de elementos visuais como WoofCard, WoofPost e inputs personalizados.

# Como rodar:

1. Instale as dependências com <b>"npm install"</b>
2. Inicie o servidor com <b>"npm run dev"</b>
